📖 Introduction

Parcel Obfuscate Bypass adalah sebuah loader yang memungkinkan script dimuat langsung melalui GitHub Raw URL. Produk ini dirancang agar proses update script menjadi lebih mudah tanpa perlu mengirim ulang file setiap kali terdapat perubahan.

---

✨ Features

- ✔️ Load script langsung dari GitHub Repository.
- ✔️ Mudah diperbarui tanpa mengubah file injector.
- ✔️ Praktis dan ringan digunakan.
- ✔️ Mendukung Roblox Studio.
- ✔️ Instalasi hanya membutuhkan beberapa langkah.

---

📦 Requirements

- Akun GitHub.
- Roblox Studio.
- Script Bypass dari @zexxell.

---

🚀 Installation Guide

Step 1 — Create a GitHub Account

Apabila belum memiliki akun GitHub, silakan daftar terlebih dahulu.

Step 2 — Create a New Repository

Buat repository baru pada akun GitHub Anda.

«Jika belum mengetahui cara membuat repository, silakan mengikuti tutorial resmi GitHub atau video YouTube berikut:

- GitHub Docs: https://docs.github.com/en/repositories/creating-and-managing-repositories/creating-a-new-repository
- YouTube: "How to create your first GitHub repository"»

Step 3 — Upload Script

Buat file baru di dalam repository Anda, kemudian paste kode berikut:

-- Bypass Script GitHub

Di bawah kode tersebut, tambahkan script Anda sendiri:

-- Your Script

Lalu simpan (Commit Changes).

---

Step 4 — Copy Raw URL

Buka file yang telah dibuat, kemudian klik Raw dan salin URL Raw GitHub.

Contoh:

https://raw.githubusercontent.com/username/repository/main/script.lua

---

Step 5 — Configure Injector

Buka Roblox Studio.

Paste Script Injector yang telah disediakan.

Kemudian ganti bagian URL Raw GitHub dengan URL Raw milik repository Anda.

Contoh:

local URL = "YOUR_RAW_GITHUB_URL"

menjadi

local URL = "https://raw.githubusercontent.com/username/repository/main/script.lua"

---

✅ Done

Apabila seluruh langkah telah dilakukan dengan benar, script akan dimuat langsung dari repository GitHub Anda.
